# Descripción
Este repositorio contiene el desarrollo del Taller T11 de la asignatura Sistemas Operativos, enfocado en el uso básico del sistema operativo Linux, manejo de comandos, jerarquía de directorios, lectura y manipulación de archivos, compilación de programas en C, gestión de procesos y administración de permisos.
El trabajo fue desarrollado de manera colaborativa utilizando Git y GitHub como sistema de control de versiones distribuido, permitiendo sincronización asincrónica entre los integrantes del equipo.

> **Descripción:** El presente taller tiene como objetivo principal fortalecer la comprensión práctica del funcionamiento básico de un sistema operativo Linux mediante el uso de la línea de comandos. Se busca que el estudiante adquiera habilidades en la creación y organización de directorios, así como en la manipulación de archivos utilizando comandos propios del entorno Unix/Linux.

Asimismo, se pretende desarrollar la capacidad de interpretar y utilizar herramientas del sistema para la lectura de archivos, el uso de pipes para el procesamiento de información y la ejecución de comandos encadenados. Otro objetivo fundamental es comprender el manejo de procesos, diferenciando entre ejecución en primer y segundo plano, así como la administración y finalización de procesos mediante comandos del sistema.

El taller también tiene como propósito reforzar los conocimientos relacionados con la compilación de programas en lenguaje C utilizando el compilador GCC, incluyendo la ejecución de programas con argumentos desde la terminal y la implementación de validaciones básicas de errores mediante llamadas al sistema. De igual manera, se busca comprender el funcionamiento de los permisos de archivos y su modificación a través del comando chmod.

Finalmente, se pretende aplicar buenas prácticas de trabajo colaborativo mediante el uso de Git y GitHub como sistema de control de versiones distribuido, permitiendo la gestión organizada de cambios, el seguimiento de versiones y la sincronización del trabajo entre los integrantes del equipo.
